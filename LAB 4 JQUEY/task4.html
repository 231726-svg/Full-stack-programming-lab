Task 04 – jQuery Topics

👋 Introduction to jQuery

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

  • Write less, do more
  • Cross-browser compatible
  • Rich plugin ecosystem
  • Current version: 3.6.0

🔧 DOM Manipulation Methods

jQuery provides powerful methods for selecting and manipulating HTML elements in the DOM tree.

  • .html() – Get or set inner HTML
  • .text() – Get or set text content
  • .append() / .prepend() – Insert content
  • .remove() / .empty() – Remove elements
  • .attr() / .prop() – Attributes & properties
  • .css() – Get or set CSS styles

⚡ Event Handling

jQuery makes event binding simple and consistent across all browsers. You can attach multiple events with a single call.

  • .on() – Attach event handlers
  • .off() – Remove event handlers
  • .click() / .dblclick() – Mouse events
  • .hover() – Mouseenter + mouseleave
  • .keypress() / .keyup() – Keyboard events
  • .submit() / .change() – Form events

🌐 Ajax with jQuery

jQuery's Ajax methods let you communicate with a server and load data into the page without a full page reload.

  • $.get() – Load data using HTTP GET
  • $.post() – Send data using HTTP POST
  • $.ajax() – Full configuration object
  • $.getJSON() – Fetch JSON data
  • Supports Promises with .done() / .fail() / .always()

🔌 jQuery Plugins

The jQuery plugin ecosystem extends its core functionality. Thousands of open-source plugins are available for common tasks.

  • jQuery UI – Interactions, widgets, effects
  • jQuery Validate – Form validation
  • DataTables – Advanced tables
  • Slick – Carousel / slider
  • Select2 – Enhanced select boxes
